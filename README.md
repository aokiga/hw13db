# hw13db
HW 13 for Databases using Kotlin and [Exposed](https://github.com/JetBrains/Exposed)     
You must insert your SQLCourseDatabase url, user and password info into `UserConfig.kt`     
## Usage
Examples:    
* `./gradlew run --args='c`
* `./gradlew run --args='g "Высшая математика" "Алиев"'`
* `./gradlew run --args='u "Высшая математика" "Алиев" 5'`
